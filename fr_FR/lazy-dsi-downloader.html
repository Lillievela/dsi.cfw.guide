Lazy DSi Downloader

Join the DS⁽ⁱ⁾ Mode Hacking! Discord server for guide development and help.

« Lazy DSi Downloader » est un outil pour simplifier le processus de modulation d'une console Nintendo DSi.

# Étapes de vérification

  1. Allumez votre Nintendo DSi
  2. Lancez l'appliaction Appareil photo Nintendo DSi

Si, à ce stade, vous obtenez un tutoriel et faites planter la console en essayant de le suivre, alors vous ne pouvez pas utiliser Memory Pit. Please see Alternate Exploits instead.

# Guide d'installation

  1. Si vous êtes sous Windows & installez la dernière version de [7-Zip](https://www. 7-zip. org/download. html)
    • Cela ne fonctionnera pas avec un autre outil d'extraction d'archives que vous possédez, tel que WinRAR. Lazy DSi File Downloader repose sur 7-Zip lui-même, et non sur un extracteur d'archive général
  2. Téléchargez la dernière version de Lazy DSi File Downloaderopen in new window pour votre OS
  3. Lancez-le via les instructions listées dans la page de publication
  4. Appuyez sur le bouton « Next »
    • La lecture du texte d'introduction n'est pas obligatoire
  5. Si vous rencontrez un problème lors des étapes de vérification, désactivez l'option Memory Pit
  6. Cochez les cases « Download GodMode9i » et « Download Unlaunch »
    • N'hésitez pas à sélectionner l'une des autres applications homebrew dans Additional homebrew..., mais ce n'est pas obligatoire
  7. Attendez que tout soit téléchargé, puis cliquez sur « Finish »
