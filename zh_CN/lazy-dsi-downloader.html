Lazy DSi File Downloader（DSi懒人下载包）

Join the DS⁽ⁱ⁾ Mode Hacking! Discord server for guide development and help.

“Lazy DSi Downloader”是一个简化破解Nintendo DSi 过程的工具。

# 核查步骤

  1. 打开你的任天堂DSi
  2. 启动到 Nintendo DSi 相机应用程序

如果此时你在尝试进行教程时崩溃，那么你就不能使用Memory Pit漏洞. Please see Alternate Exploits instead.

# 设置指南

  1. 如果您使用Windows，请下载 & 安装最新版本的 7-Zipopen in new window
    • 您拥有的任何其他解压工具将不适用，如WinRAR。 The Lazy DSi File Downloader relies on 7-Zip itself, and not a general archive extractor
  2. 下载最新版本的 [Lazy DSi File Downloader](https://github. com/yourkalamity/lazy-dsi-file-downloader/releases)
  3. 按照发布页面中列出的说明启动Lazy Dsi Downloader
  4. 点击Next按钮
    • Intro text不是必须阅读的
  5. 如果您的验证步骤出现崩溃，请切换Memory Pit 选项
  6. 点击"Download GodMode9i" 和 "Download Unlaunch"的复选框
    • 在Additional homebrew...中自由选择自制程序, 但这不是必须的
  7. 等待所有下载，然后点击完成
